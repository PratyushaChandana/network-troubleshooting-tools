# Network Troubleshooting Toolkit 

A collection of lightweight Bash automation utilities engineered for rapid network infrastructure diagnostics, connectivity testing, and DNS verification.

##  CompTIA A+ Alignment
*   **Domain 3.0 (Software Troubleshooting):** This repository implements automated testing templates to rapidly diagnose client-side network failures, separating localized physical layer faults from wide-area name resolution issues.

##  Core Diagnostic Scripts
*   `check_ping.sh` — Automates ICMP echo requests to confirm basic end-to-end network reachability.
*   `check_dns.sh` — Wraps the `nslookup` utility to test localized and public domain name configuration integrity.
